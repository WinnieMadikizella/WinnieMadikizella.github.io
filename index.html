Nashville Housing Data Cleaning Project

This project showcases my expertise in SQL for data cleaning and preparation. I worked with the Nashville Housing dataset to transform raw data into a structured and analyzable format. Key tasks included standardizing date formats, populating missing addresses using self-joins, splitting and extracting address components, updating categorical values, removing duplicate records, and optimizing the dataset by dropping unused columns. This project demonstrates my ability to clean and preprocess data efficiently for analysis and decision-making.

Covid 19 Data Exploration in SQL

This project involves an in-depth exploration of COVID-19 death and vaccination data, focusing on data cleaning and analysis to extract key insights. The analysis covers important metrics such as death rates, infection rates, and vaccination progress across different locations. By leveraging SQL, the project examines the ratio of deaths to cases, tracks infection rates, and measures the cumulative number of vaccinations. Additionally, global and continental trends are analyzed to provide a comprehensive view of the pandemic's impact.
